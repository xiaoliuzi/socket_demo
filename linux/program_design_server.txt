/* Block socket verison */
1）初始化socket地址结构
2）创建连接套接字、监听套接字
3）将套接字与地址结构绑定
4）开始监听套接字
5）等待客户端的连接请求
6）如果有新的连接请求，继续下一步;否则，跳回第5步
7）接受新的连接请求   <=== accept函数
8）从连接套接字读取一个字节的数据
9）如果读到的数据不是EOF文件结束符,继续下一步;否则跳转至第13步
10）如果读完全部数据，继续下一步;否则,跳回第8步
11）将读到的数据发送给客户端
12）跳回至第8步  <=== accpet函数，在没有新连接到来时，是阻塞的;
				<==== 跳回第8步，防止accept阻塞
13）关闭连接套接字
14）关闭监听套接字
15）程序结束



/* No block socket version */
1）初始化socket地址结构、套接字数组
2）创建连接套接字、监听套接字,并设置连接套接字为非阻塞模式
3）将套接字与地址结构绑定
4）开始监听套接字
5）如果有新的建立连接请求，则继续下一步;否则，跳至第8步
6）如果连接成功建立，则继续下一步;否则，跳至第8步
7）依次将新建立的连接的套接字存储到，连接请求描述符数组中
8）从数组第0个元素开始，逐一查询连接请求数组中的每一个描述符读取字符数，如果从套接子中读取字符数不为0，则继续下一步;否则，继续读取下一个字符数，把当前计数的所有已连接套接字全部查询完毕
9）如果从连接套接字中读取字符数不为0，则将读取的字符原样写回套接字中。否则，跳回第5步


13）关闭连接套接字
14）关闭监听套接字
15）程序结束
